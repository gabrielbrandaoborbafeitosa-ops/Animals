#Animals
Animals é um website para digitar um animal, e, o som dele vai aparecer como texto, existem animais mais difíceis de encontrar, que, nesse caso, o website mostra.
Exemplos:
Vaca;
Abelha;
Galo;
Cachorro,
Gato
